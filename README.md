# dotfiles-editors

Editor configs: `nvim/`.

Part of the [dotfiles umbrella](https://github.com/blak0p/dotfiles).

## Standalone install

```bash
git clone https://github.com/blak0p/dotfiles-editors.git
cd dotfiles-editors
./install.sh
```

The installer symlinks `nvim/` into `~/.config/nvim`.

## What's deployed

- `nvim` → `~/.config/nvim` (Neovim config)
